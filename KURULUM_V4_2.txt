
# V4.2 Smart Report Automation Pack

Bu paket v4.1.0 üzerine kopyalanır.

## Amaç

Multi-machine site raporunu otomasyona hazır hale getirmek, site rapor geçmişini eklemek ve dashboard’dan rapor üretip kaydetmeyi sağlamak.

## Eklenenler

```text
Site rapor geçmişi API
Site rapor detay API
Dashboard’dan site raporu oluştur + kaydet butonu
Son site raporu görünümü
Site rapor geçmişi görünümü
Site rapor detay görünümü
n8n Smart Report Automation workflow
```

## Kurulum

ZIP'i proje ana klasörünün üzerine çıkar:

```text
C:\New DeviceOs Project\MiaDeviceOS_v0.5_command_engine (1)
```

"Dosyaları hedefte değiştir" seçeneğini onayla.

## Backend Yeniden Başlatma

```powershell
cd "C:\New DeviceOs Project\MiaDeviceOS_v0.5_command_engine (1)\platform\backend"
npm.cmd start
```

Eğer port hatası olursa:

```powershell
netstat -ano | findstr :3100
taskkill /PID <PID> /F
npm.cmd start
```
