# V4.7 SaaS Admin Panel Pack

Bu paket v4.6.1 üzerine kopyalanır.

## Amaç

Signup ile oluşan kullanıcı, customer, site ve tenant access kayıtlarını dashboard üzerinden görmeye başlamak.

## Eklenenler

```text
Admin panel ekranı
GET /api/admin/overview
GET /api/admin/users
GET /api/admin/customers
GET /api/admin/sites
GET /api/admin/tenant-access
Dashboard’a Admin Panel linki
Admin listeleri için yardımcı index migration
```

## Kurulum

ZIP'i proje ana klasörünün üzerine çıkar:

```text
C:\New DeviceOs Project\MiaDeviceOS_v0.5_command_engine (1)
```

"Dosyaları hedefte değiştir" seçeneğini onayla.

## Backend Yeniden Başlatma

```powershell
cd "C:\New DeviceOs Project\MiaDeviceOS_v0.5_command_engine (1)\platform\backend"
npm.cmd start
```
