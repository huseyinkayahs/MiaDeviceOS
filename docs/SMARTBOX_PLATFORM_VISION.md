# SmartBox Platform Vision

## Amaç
SmartBox Platform, küçük ve orta ölçekli üretim işletmeleri için modüler veri toplama, otomasyon ve AI raporlama altyapısıdır.

---

## Platform Bileşenleri

```text
SmartBox Core
SmartFlows
SmartAI
SmartVision
SmartDashboard
```

### SmartBox Core
Sahadaki makinelerden veri toplayan cihaz katmanı.

### SmartFlows
Hazır n8n otomasyon paketleri.

Örnekler:

```text
Üretim takibi
Enerji takibi
Bakım uyarısı
Depo / stok bildirimi
Makine duruş bildirimi
```

### SmartAI
Verileri yorumlayan ve doğal dilde raporlayan analiz katmanı.

### SmartVision
Kamera ile sayım, kalite kontrol ve görsel denetim katmanı.

### SmartDashboard
Web ve mobil yönetim paneli.

## Ticari Model Taslağı

Örnek fiyatlandırma:

```text
Kurulum: 15.000 TL
Donanım: 8.000 TL
Aylık yazılım: 1.500 TL
```

Hedef:

```text
50 müşteri → 75.000 TL / ay abonelik
200 müşteri → 300.000 TL / ay abonelik
```

Not:
Bu rakamlar ilk ticari planlama içindir. Gerçek fiyatlandırma pilot müşteri geri bildirimine göre netleşecektir.
