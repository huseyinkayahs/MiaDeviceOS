# FactoryBox V2 to V3 Plan

## V2 result

V2 produced a pilot-ready software and workflow foundation.

## V3 direction

V3 should move from simulated/bench validation to real hardware and field validation.

Recommended order:

```text
v3.1 Real DI1 Hardware Input Module
v3.2 Pilot Bench Wiring Test
v3.3 Real Current Sensor Driver
v3.4 Current Sensor Calibration
v3.5 First Machine Pilot Installation
v3.6 Pilot Feedback and Reliability Improvements
```

## V3 principle

Do not add many new features before the first real machine test.

The priority is:

```text
Reliable signal input
Safe panel wiring
Correct RUN/STOP detection
Useful daily report
Useful stop alert
```
