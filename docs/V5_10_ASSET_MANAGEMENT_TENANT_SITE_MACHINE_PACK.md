# FactoryBox One v5.10.0

## Sprint Adı

v5.10.0 Asset Management / Tenant Site Machine Pack

## Amaç

FactoryBox SaaS admin panelinde tenant, site ve makine kayıtlarını daha kontrollü ve düzenli yönetilebilir hale getirmek.

## Kapsam

```text
Customer oluşturma
Customer ad / status düzenleme
Site oluşturma
Site ad / lokasyon / status düzenleme
Machine oluşturma
Machine ad / tip / status düzenleme
Assets / Machines menüsü
Machine listesi
Site limitinin abonelik kotasına bağlanması
Asset işlemlerinin audit log'a yazılması
```

## Not

Bu sprint ödeme veya cihaz firmware entegrasyonu eklemez. Ama cihaz provisioning için gerekli tenant/site/machine kayıtlarını admin panelden daha rahat yönetilebilir hale getirir.
