
# V4.9 Audit Log / Activity History Pack

Bu paket v4.8.0 üzerine kopyalanır.

## Amaç

Admin panelde yapılan değişiklikleri kayıt altına almak.

## Eklenenler

```text
admin_audit_logs tablosu
GET /api/admin/audit-logs
Kullanıcı rol/status değişikliğinde audit kayıt
Customer status değişikliğinde audit kayıt
Site status değişikliğinde audit kayıt
Admin panelde Activity History bölümü
Overview içinde Audit Logs sayacı
```

## Kurulum

ZIP'i proje ana klasörünün üzerine çıkar:

```text
C:\New DeviceOs Project\MiaDeviceOS_v0.5_command_engine (1)
```

"Dosyaları hedefte değiştir" seçeneğini onayla.

## Backend Yeniden Başlatma

```powershell
cd "C:\New DeviceOs Project\MiaDeviceOS_v0.5_command_engine (1)\platform\backend"
npm.cmd start
```
