# V3.5 Platform Backend MVP

Bu sprintte v3.5, v3.6 ve v3.7 kapsamı birleştirilmiştir.

## Amaç

FactoryBox cihazından gelen MQTT mesajlarını dinleyen, PostgreSQL’e kaydeden ve dashboard/API üzerinden gösteren ilk platform backend MVP’yi oluşturmak.

## Ön Koşul

PostgreSQL çalışıyor olmalı:

```powershell
cd "C:\New DeviceOs Project\MiaDeviceOS_v0.5_command_engine (1)\platform\database"
docker compose up -d
```

## Backend Kurulum

Yeni PowerShell aç:

```powershell
cd "C:\New DeviceOs Project\MiaDeviceOS_v0.5_command_engine (1)\platform\backend"
copy .env.example .env
npm.cmd install
npm.cmd start
```
