
# V4.0 Multi Machine Daily Report

Bu paket v3.9.0 üzerine kopyalanır.

## Amaç

Aynı site altındaki tüm makineleri tek günlük yönetici raporunda toplamak.

## Eklenenler

```text
Site günlük rapor endpointi
Site Telegram rapor endpointi
Genel fabrika skoru
Makine bazlı skorlar
Dashboard Multi-Machine Günlük Yönetici Raporu alanı
n8n Telegram workflow taslağı
Site rapor index migration dosyası
```

## Kurulum

ZIP'i proje ana klasörünün üzerine çıkar:

```text
C:\New DeviceOs Project\MiaDeviceOS_v0.5_command_engine (1)
```

"Dosyaları hedefte değiştir" seçeneğini onayla.

## Backend Yeniden Başlatma

```powershell
cd "C:\New DeviceOs Project\MiaDeviceOS_v0.5_command_engine (1)\platform\backend"
npm.cmd start
```

Eğer port hatası olursa:

```powershell
netstat -ano | findstr :3100
taskkill /PID <PID> /F
npm.cmd start
```
