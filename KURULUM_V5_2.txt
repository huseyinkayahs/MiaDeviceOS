
# V5.2 Role Based Permission Pack

Bu paket v5.1.0 üzerine kopyalanır.

## Amaç

Owner / admin / operator / viewer rollerini gerçek yetkilere bağlamak.

## Eklenenler

```text
Role permission matrix
Backend permissionRequired middleware
Admin panel Permission Overview
Admin panel butonlarını role göre gösterme/gizleme
User role/status endpointlerinde MANAGE_USERS kontrolü
Customer status endpointinde MANAGE_CUSTOMERS kontrolü
Site status endpointinde MANAGE_SITES kontrolü
Invite endpointlerinde MANAGE_INVITES kontrolü
Audit log endpointinde AUDIT_VIEW kontrolü
Report üretim/gönderim endpointlerinde SEND_REPORTS kontrolü
```

## Rol Yetki Matrisi

```text
system_admin:
  tüm yetkiler

owner:
  ADMIN_VIEW
  MANAGE_USERS
  MANAGE_CUSTOMERS
  MANAGE_SITES
  MANAGE_INVITES
  AUDIT_VIEW
  SEND_REPORTS
  VIEW_REPORTS
  VIEW_DASHBOARD

admin:
  ADMIN_VIEW
  MANAGE_CUSTOMERS
  MANAGE_SITES
  MANAGE_INVITES
  AUDIT_VIEW
  SEND_REPORTS
  VIEW_REPORTS
  VIEW_DASHBOARD

operator:
  SEND_REPORTS
  VIEW_REPORTS
  VIEW_DASHBOARD

viewer:
  VIEW_REPORTS
  VIEW_DASHBOARD
```

## Kurulum

ZIP'i proje ana klasörünün üzerine çıkar:
