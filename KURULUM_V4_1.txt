
# V4.1 Device Info Sync

Bu paket v4.0.0 üzerine kopyalanır.

## Amaç

Database içindeki eski device firmware bilgisini gerçek MQTT / command status payloadlarından gelen bilgilerle güncellemek.

## Eklenenler

```text
Device info schema genişletme
MQTT payloadlarından firmware/model/platform/build sync
Device info API endpointleri
Dashboard Device Info kartı
Multi-machine report center içinde doğru device alanları
```

## Kurulum

ZIP'i proje ana klasörünün üzerine çıkar:

```text
C:\New DeviceOs Project\MiaDeviceOS_v0.5_command_engine (1)
```

"Dosyaları hedefte değiştir" seçeneğini onayla.

## Backend Yeniden Başlatma

```powershell
cd "C:\New DeviceOs Project\MiaDeviceOS_v0.5_command_engine (1)\platform\backend"
npm.cmd start
```

Eğer port hatası olursa:

```powershell
netstat -ano | findstr :3100
taskkill /PID <PID> /F
npm.cmd start
```

## Not

Eğer firmware hâlâ eski görünüyorsa, cihazdan firmware_version içeren bir MQTT mesajı gelmemiş demektir.
Bu durumda MQTT Explorer’dan cihazın get_device_info / get_health / get_diagnostics gibi komutlarından biri gönderilip command/status cevabı beklenebilir.
