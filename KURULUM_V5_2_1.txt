
# V5.2.1 User Permission Visibility Fix

Bu paket v5.2.0 üzerine kopyalanır.

## Amaç

Admin panelde sadece aktif kullanıcının değil, diğer kullanıcıların rollerine göre hangi yetkileri aldığını da net göstermek.

## Eklenenler

```text
Role Matrix tablosu
Kullanıcılar tablosunda Permissions kolonu
Role değişince permission önizlemesinin canlı değişmesi
Permission Overview daha açıklayıcı hale getirildi
```

## Kurulum

ZIP'i proje ana klasörünün üzerine çıkar:

```text
C:\New DeviceOs Project\MiaDeviceOS_v0.5_command_engine (1)
```

"Dosyaları hedefte değiştir" seçeneğini onayla.

## Backend Yeniden Başlatma

```powershell
cd "C:\New DeviceOs Project\MiaDeviceOS_v0.5_command_engine (1)\platform\backend"
npm.cmd start
```
