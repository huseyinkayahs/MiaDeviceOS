
# V4.4 OpenAI SmartAI Upgrade

Bu paket v4.3.0 üzerine kopyalanır.

## Amaç

Kural bazlı SmartAI site raporunu OpenAI destekli daha doğal yönetici yorumuna yükseltmek.

## Eklenenler

```text
OpenAI status endpoint
OpenAI destekli site raporu endpointi
OpenAI Telegram format endpointi
Dashboard OpenAI SmartAI Upgrade alanı
OpenAI Raporu Oluştur + Kaydet butonu
Rule fallback mekanizması
.env OpenAI örnek dosyası
```

## Kurulum

ZIP'i proje ana klasörünün üzerine çıkar:

```text
C:\New DeviceOs Project\MiaDeviceOS_v0.5_command_engine (1)
```

"Dosyaları hedefte değiştir" seçeneğini onayla.

## .env Ayarı

Backend klasöründeki `.env` dosyana ekle:

```text
OPENAI_API_KEY=BURAYA_API_KEY
OPENAI_MODEL=gpt-5-mini
SMARTAI_OPENAI_ENABLED=true
```

API key'i asla GitHub'a gönderme.

## Backend Yeniden Başlatma

```powershell
cd "C:\New DeviceOs Project\MiaDeviceOS_v0.5_command_engine (1)\platform\backend"
npm.cmd start
```

Eğer port hatası olursa:

```powershell
netstat -ano | findstr :3100
taskkill /PID <PID> /F
npm.cmd start
```
