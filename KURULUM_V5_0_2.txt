
# V5.0.2 Invite Accept User Insert Fix

Bu paket v5.0.1 üzerine kopyalanır.

## Düzeltilen Hata

Davet kabul ekranında şu hata oluşuyordu:

```text
null value in column "id" of relation "app_users" violates not-null constraint
```

## Kök Sebep

Davet kabul sırasında yeni kullanıcı oluşturulurken `app_users.id` alanı doldurulmuyordu.

Aynı blokta ayrıca FactoryBox'ın mevcut şifre yapısı yerine bcrypt kullanılmaya çalışılmıştı.

## Düzeltme

```text
app_users.id için makeUserId() kullanıldı
password_salt alanı dolduruldu
bcrypt yerine mevcut makeSalt() + hashPassword() yapısına dönüldü
Davet kabul sonrası session/token oluşturma düzeltildi
```
