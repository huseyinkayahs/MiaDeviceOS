
# V5.0.3 Invite Session Dashboard Fix

Bu paket v5.0.2 üzerine kopyalanır.

## Düzeltilen Durum

Davet kabul edildikten sonra dashboard açılıyordu fakat SaaS Foundation alanları boş görünüyor ve üstte "Hata" yazıyordu.

## Kök Sebep

Dashboard tüm API çağrılarını bitirdikten sonra login/tenant bilgisini ekrana yazıyordu.

Davetli kullanıcı customer/site yetkisiyle geldiğinde bazı site API'leri erişim/veri hatası verince ekran komple hata moduna geçiyor, bu yüzden login bilgisi de görünmüyordu.

## Düzeltme

```text
Login/tenant bilgisi önce render edilir
Dashboard siteCode tenant current_site üzerinden dinamik seçilir
Bazı site/machine API çağrıları hata verirse tüm ekran bozulmaz
Customer-level davetlerde tenant altındaki siteler genişletilir
Site access kontrolü customer-level erişimi de dikkate alır
```
